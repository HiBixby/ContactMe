설명

* HiBixby와 연락을 하기 위한 Vue 웹사이트
* 댓글을 쓰면 개발자의 디스코드 채널에 webhook 형태로 게시됨 (개발자만 볼수 있음)
* 게시된 webhook은 개발자의 데스크톱,노트북,핸드폰,아이패드 등에 동시에 디스코드 알림을 보내준다.

만든 이유
* 레포 주인이 디스코드 알림은 자주 보기 때문이다.
